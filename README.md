## X-mas tree

### Description
Simple christmas tree.
For single developers only.

### How to?
1. clone, build and execute
    ```bash
    git clone https://github.com/minsoo0926/xmas-tree.git
    cd xmas-tree
    mkdir build && cd build && cmake .. && make xmas
    ./xmas
    ```
    1-2. (OR...) Download execute from release, matching to your OS
3. Enjoy your christmas!
